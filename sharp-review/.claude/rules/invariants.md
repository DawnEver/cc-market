# Sharp Review Invariants

Dev-only constraints on `scripts/sharp-review-workflow.js` and `lib.mjs` — not restating
the agent-facing flow already in `skills/sharp-review/SKILL.md` (and `reference/
task-system.md`), which is the runtime source of truth and must not drift from these.

## Workflow script
- All Workflow args originate from `diff-manifest.js`'s output: `{ date, mode, range, stats, diff?, manifestText?, excludedSummary }`. `date` is YYYY-MM-DD — never call `Date.now()`/`new Date()` inside `sharp-review-workflow.js` (breaks resume caching).

## Schema
- StructuredOutput schema must be `{ type: 'object', properties: { findings: [...] } }`. Bare array schema causes silent failure.

## Finding IDs
- Format: `SR-YYYYMMDD-NNN`. Assigned sequentially by the workflow merge phase — never generated by agents.
